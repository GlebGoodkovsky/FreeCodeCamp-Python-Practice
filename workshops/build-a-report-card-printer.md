# Build a Report Card Printer (Workshop)

## Code

```python
name = 'Alice'
print(name, type(name))

is_student = True
print(is_student, type(is_student))

age = 20
print(age, type(age))

score = 80.5
print(score, type(score))

print(isinstance(score, float))
```

## Output

```
Alice <class 'str'>
True <class 'bool'>
20 <class 'int'>
80.5 <class 'float'>
True
```
